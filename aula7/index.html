Aula 07 -estudos de CSS

A sigla CSS éCascading Style Sheets é uma linguagem ultilizada para estilizarpágina da web.Utilizando esta linguagem podemos definir.

  • Cor de fundo,e elemento.
  • Fonte e tamanho das letras
  • Posicionamento de elementos dentro da página
  • Efeitos visuais e animações

O CSS funciona em conjunto com o HTML, enquanto o HTML diz o que aparece, o CSS diz como deve aparecer.
